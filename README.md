# Credit_card_fraud_detection
End-to-end Credit Card Fraud Detection using Python, SMOTE, Random Forest, Streamlit, and Power BI Dashboard with 99.95% accuracy and 95.2% ROC-AUC.

## 📌 Project Overview

This project aims to detect fraudulent credit card transactions using Machine Learning. Due to the highly imbalanced nature of the dataset, SMOTE was used to balance the classes before training a Random Forest model.

## 🚀 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Random Forest
* SMOTE
* Streamlit
* Power BI
* Matplotlib
* Seaborn

## 📊 Dataset

* Credit Card Transactions Dataset
* Duplicate records removed
* Highly imbalanced dataset handled using SMOTE

## 🔍 Models Used

* Logistic Regression
* Random Forest (Final Model)

## 📈 Model Performance

| Metric    | Value  |
| --------- | ------ |
| Accuracy  | 99.95% |
| Precision | 91%    |
| Recall    | 75%    |
| F1 Score  | 82%    |
| ROC-AUC   | 95.2%  |

## 📌 Key Results

* Successfully detected 71 fraudulent transactions.
* Generated only 7 false alarms.
* Achieved excellent ROC-AUC score of 95.2%.
* Improved fraud detection performance using SMOTE.

## 📊 Power BI Dashboard

Interactive dashboard created to visualize:

* Transaction distribution
* Fraud distribution
* Model performance metrics
* Confusion matrix

## 🌐 Streamlit App

Developed a Streamlit web application for predicting whether a transaction is genuine or fraudulent.
